ServletContext?
https://java117.tistory.com/18
- ServletContext 클래스는 톰캣 컨테이너 실행 시 각 컨텍스트(웹 애플리케이션)마다 한 개의 ServletContext객체를 생성합니다.
- 그리고 톰캣 컨테이너가 종료하면 ServletContext객체 역시 소멸됩니다.
- 서블릿과 컨테이너 간의 연동을 위해 사용한다.(컨테이너...가..뭐지?ㅜㅜ)
- 서블릿끼리 자원 공유하는데 사용한다.
- 컨테이너 실행시 생성되고 컨테이너 종료시 소멸된다.

ServletContext가 제공하는 기능?
- 서블릿에서 파일 접근 기능
- 자원 바인딩 기능
- 로그 파일 기능
- 컨텍스트에서 제공하는 설정 정보 제공 기능
