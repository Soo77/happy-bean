▶ MemberController 만들 때 클래스위에 @Controller랑 @RequestMapping은 뭘까?
- @Controller 어노테이션은 Spring MVC 프레임워크에서 사용되는 주석이고, 특정 클래스가 컨트롤러 역할을 수행함을 나타낸다.
- @Controller를 붙이면 해당 클래스를 웹 요청을 처리하는 컨트롤러로 사용할 수 있다.
- @RequestMapping 어노테이션은 URL을 컨트롤러의 메서드와 매핑할 때 사용하는 스프링 프레임워크의 어노테이션이다.
클래스나 메서드 선언부에 @RequestMapping과 함께 URL을 명시하여 사용한다.

▶ SYSDATE() 랑 NOW()의 차이점..?
- sysdate()는 예측불가능하다. 요 함수가 호출될 때마다 매번 호출 시점의 시각을 재계산하여 리턴한다. 반면 now()는 포함된 쿼리문이 최초 실행될 때의 시각을 리턴한다.
- now()의 경우 한 쿼리문에서 항상 같은 값을 리턴한다.

▶ java.util.Date와 java.sql.Date 차이
- java.util.Date 는 시간, 초의 세부 정보를 포함하고 있음
- java.sql.Date 는 시간, 초의 정보는 포함하고 있지 않음 - 또한 java.util.Date를 상속받은 클래스이다.
